Timestamp Microservice

How does it work?

Input a valid date (YYYY-MM-DD) or number (number of milliseconds since Jan 1, 1970) into the input box and hit return and you will be sent the unix timestamp of the input date in milliseconds as well as a string of the date in the format Thu, 01 Jan 1970 00:00:00 GMT. If no input is typed and submit is clicked then will display current time.

Examples

2015-12-25 1451001600000
